Preface

Applied mathematics for robust external memory in LLM systems

This course is built from a simple pressure: a language model with external memory must represent, retrieve, rank, compress, update, forget, and evaluate information. Each lecture introduces a mathematical object because a previous object fails at one of those jobs.

Vertical depth Base case, derivation, invariant, failure mode.
Horizontal application Representation, retrieval, indexing, memory, evaluation.
No coding assignments Math-first problems and design reasoning.
