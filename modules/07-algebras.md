# Module 07: Algebras - When Objects Multiply

## Core Question

What if vectors can also multiply each other?

## Limitation

Vector spaces allow addition and scalar multiplication, but not necessarily multiplication between vectors. Some systems need internal interaction.

## New Object

An algebra over a field: a vector space with a multiplication operation compatible with the vector space structure.

## Legal Operations

- Add objects.
- Scale objects.
- Multiply objects.
- Study commutativity, associativity, and identities.
- Represent abstract multiplication using matrices or operators.

## Invariants

- Dimension as a vector space
- Multiplication table
- Center
- Ideals
- Commutativity or noncommutativity
- Associativity

## Concrete Example

The quaternions have basis 1, i, j, k and multiplication rules:

```text
i^2 = j^2 = k^2 = ijk = -1
ij = k
ji = -k
```

The price of richer multiplication is that order matters.

## Hand Exercises

1. Use the quaternion rules to compute ji and ij.
2. Explain why quaternions are not a field under the usual field definition.
3. Give an example of a familiar algebra: matrices, polynomials, or complex numbers.

## Depends On

Vector spaces, fields, and multiplication rules.

## Supports Later
