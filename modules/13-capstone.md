# Module 13: Capstone - Design a Mathematical Object

## Core Question

Can you explain a mathematical object by the expressive limitation it solves?

## Task

Choose one object from the course or from a later interest area. Build an object card and a short lecture around it.

Possible choices:

- Complex numbers
- Quaternions
- Inner product spaces
- Tensor products
- Dual spaces
- Groups
- Representations
- Categories
- Lie algebras
- Hilbert spaces

## Required Object Card

```text
Object:

Limitation:

New expressive capability:

Legal operations:

Invariants:

Smallest concrete example:

One non-example:

Later theories that depend on it:

Design move:
```

## Final Explanation

Write a 1-2 page mini-lecture that begins with a limitation, introduces the new object, works through one concrete example, and ends by naming what later theory becomes possible.

## Hand Exercises

1. Create the object card.
2. Work one example by hand.
3. Write three sentences comparing the object to the object that came immediately before it.
4. Decide whether the object enlarges space, enriches scalars, invents an operation, or combines the moves.

## Completion Standard

You are done when a future version of you could read the card and understand why the object had to exist.
